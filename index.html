Mood Prediction Using Spotify Audio Features

Can machines detect the mood of a song?

This project classifies songs as "happy" or "not happy" using Spotify audio features such as valence, energy, and danceability.

Motivation

Music influences emotion in powerful ways. This project explores whether machine learning can detect emotional tone using only audio characteristics.

Dataset Features

  • Danceability
  • Energy
  • Tempo
  • Acousticness
  • Valence (used to create the target label)

Target Label

  • 1 = Happy (valence > 0.5)

Modeling

Trained and compared:

  • Logistic Regression
  • Random Forest
  • Support Vector Machine (SVM)
  • XGBoost

Results

Random Forest performed best with 72% accuracy and F1 score.

Future Improvements

  • Model valence as regression or multi-class
  • Incorporate genre, lyrics, or user data
  • Explore deep learning on spectrograms
